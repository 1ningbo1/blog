# 欢迎使用 Nb's Blog

这是一篇示例文章，展示了博客系统的基本功能。

## 什么是 Nb's Blog？

Nb's Blog 是一个轻量级的博客系统，基于 GitHub Pages 和原生 Web 技术构建，无需后端服务器即可运行。
